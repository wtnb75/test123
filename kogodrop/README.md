# 古語ドロップ (KogoDrop)

落として覚える古語単語ゲーム。高校受験対策向けの古語・現代語・英語を同時に学べる落ちゲー。

## ゲームの目的

上から落ちてくる単語タイルを、画面下部に並ぶ意味スロット（4列）へ正しく振り分ける。  
全問終了後に正答率を確認し、間違えた単語を復習できる。

## ルール

- 落下タイルに表示された単語を、対応する意味のスロットへ落とす
- 正解: 正答数 +1
- 不正解: 不正答数 +1（ゲームは続行）
- 規定出題数に到達したらリザルト画面へ
- ゲームオーバーなし（時間制限なし）

### 言語モード

| モード | タイル | スロット |
|--------|--------|---------|
| 古語→現代語 | 古語（例: あはれ） | 現代語訳の短縮形 |
| 古語→英語 | 古語 | 英語訳の短縮形 |
| 英語→古語 | 英語訳の短縮形 | 古語 |

### 難易度

| 難易度 | 出題プール |
|--------|-----------|
| やさしい | basic 語（約60語） |
| ふつう | basic + standard（約120語） |
| むずかしい | 全レベル（約160語） |

## 操作方法

### タッチ操作
- タイルをタッチしてホールド → 左右にドラッグ → 離してスロットへ投入

### キーボード操作
- `←` / `→`: タイルを左右のスロットへ移動
- `Space` / `Enter`: 現在位置のスロットへ投入
- `P`: 一時停止（未実装 / 非MVP）

## 開発

```bash
npm install
npm run dev       # 開発サーバー起動 (port 8080)
npm run build     # 本番ビルド
npm run test      # テスト実行
npm run test:coverage  # カバレッジ付きテスト
npm run lint      # ESLint
```

### 開発用フラグ

`.env.development` に以下を記述すると `verified: false` の単語も出題プールに含まれる:

```
VITE_DEV_IGNORE_VERIFIED=true
```
